HTML – Struktura

  • HyperText Markup Language
  • Ndërton strukturën e faqes
  • Tituj, paragrafë, imazhe, linke
<h1>Mirë se vini</h1>
<p>Ky është një paragraf.</p>
<a href="#">Lexo më shumë</a>

HTML është skeleti i çdo website-i.

CSS – Stili

  • Ngjyra
  • Fonte
  • Dizajn
h1 {
  color: #023AF9;
  font-family: Inter, sans-serif;
}
.card {
  border-radius: 12px;
}

CSS e bën strukturën të bukur.

JavaScript – Programim

  • Gjuhë programimi
  • Interaktivitet
  • Logjikë

const btn = document.querySelector('.btn');
btn.addEventListener('click', () => {
  alert('Website aktiv!');
});
